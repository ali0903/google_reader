## 全世界最美的铁路线（组图）
### by 徐铁人
### at 2011-08-08 06:19:43
### original <http://blog.sina.com.cn/s/blog_4acf0a970102drw7.html>

<p><a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s5.sinaimg.cn/orignal/4acf0a97ga9f1be99cbb4"><img name="image_operate_2191312757007117" src="http://s5.sinaimg.cn/middle/4acf0a97ga9f1be99cbb4&amp;690" width="640" height="427"></a><br>
在瑞士中部拥有诸多的湖泊与山脉，而蜿蜒于其中的一段铁路堪称全球最美的铁路线之一了。或许正是因为沿线的景色实在过于优美瑞士人特地给这条线路起了一个别称：黄金线。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s10.sinaimg.cn/orignal/4acf0a97ga9f1bf60e369"><img name="image_operate_60811312757107644" src="http://s10.sinaimg.cn/middle/4acf0a97ga9f1bf60e369&amp;690" width="640" height="427"></a><br>
这段线路景色优美，同样行驶在这段铁路上的火车也别具特点，乍一走入真有种回到了上个世纪初的感觉，强烈的怀旧风格使得乘坐这段铁路线成为了视觉上的饕餮之旅。此图为其中的二等车厢，说实话刚一走入的时候竟以为这是头等车厢。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s16.sinaimg.cn/orignal/4acf0a97ga9f1c0243aef"><img name="image_operate_52991312757154818" src="http://s16.sinaimg.cn/middle/4acf0a97ga9f1c0243aef&amp;690" width="640" height="427"></a><br>
二等车厢里的一个小包间。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s11.sinaimg.cn/orignal/4acf0a97ga9f1c9dfc8aa"><img name="image_operate_47181312757188842" src="http://s11.sinaimg.cn/middle/4acf0a97ga9f1c9dfc8aa&amp;690" width="640" height="427"></a><br>
二等车厢里的老式台灯，让我想起了刚刚热播过的电视剧《潜伏》。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s4.sinaimg.cn/orignal/4acf0a97g7764faa90b73"><img name="image_operate_80721312757222881" src="http://s4.sinaimg.cn/middle/4acf0a97g7764faa90b73&amp;690" width="640" height="427"></a><br>
这才是头等车厢，明细的区别就是这截车厢里的木质结构用的都是上等木料，座椅也是软式的沙发椅。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s5.sinaimg.cn/orignal/4acf0a97g7764fab22e74"><img name="image_operate_95901312755887901" src="http://s5.sinaimg.cn/middle/4acf0a97g7764fab22e74&amp;690" width="640" height="427"></a><br>
车厢的窗户也保留了百年前火车车窗的款式，其中中间的一块玻璃还是突出出去的，这样的设置或许是为了更好地观景吧。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s16.sinaimg.cn/orignal/4acf0a97ga9f1cbe48fbf"><img name="image_operate_85941312757341114" src="http://s16.sinaimg.cn/middle/4acf0a97ga9f1cbe48fbf&amp;690"></a><br>
一等车厢里，一条安静的大狗-快到目的地的时候这条大狗偷偷地放了个臭屁，熏得头等车厢里的乘客纷纷躲到了二等车厢，搬到二等车厢之后有个亚洲面孔的女士自嘲地评论道：其实二等车厢也是十分舒适的。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s10.sinaimg.cn/orignal/4acf0a97g7764fb2aebb9"><img name="image_operate_74781312755950898" src="http://s10.sinaimg.cn/middle/4acf0a97g7764fb2aebb9&amp;690" width="640" height="427"></a><br>
头等车厢里考究的设施。如果你持有的是欧洲铁路通票的话也可以体验一番黄金线的怀旧列车，只是需要补交一点订座费；而如果你持有的是瑞士火车通票的话则可以随时上下车而无需预订，也不需要交任何费用。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s12.sinaimg.cn/orignal/4acf0a97ga9f1d00f335b"><img name="image_operate_72651312756361015" src="http://s12.sinaimg.cn/middle/4acf0a97ga9f1d00f335b&amp;690">
</a><br>
车厢里做工考究的衣架-如果只是车厢的内部设施十分讲究的话还称不上是什么“黄金线”，真正让这条铁路线名闻遐迩的当然是车窗之外了。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s5.sinaimg.cn/orignal/4acf0a97g7764fdd94f14"><img name="image_operate_84931312757807321" src="http://s5.sinaimg.cn/middle/4acf0a97g7764fdd94f14&amp;690" width="640" height="427"></a><br>
我从卢塞恩（也称琉森）出发的时候天气极其恶劣，一连几天的雨雾天气让我的这段黄金线旅程蒙上了阴霾。此图为著名的琉森湖，清晨出发的时候虽然还淅淅沥沥地下着小雨，而湖面上似乎已经有了某种放晴的迹象。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s8.sinaimg.cn/orignal/4acf0a97ga9f1ead3f087"><img src="http://s8.sinaimg.cn/middle/4acf0a97ga9f1ead3f087&amp;690" width="640" height="427"></a><br>
一道霞光穿透了云层照射到了湖面以及岸边茂密的森林。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s6.sinaimg.cn/orignal/4acf0a97g7764fdec5915"><img name="image_operate_36581312757936161" src="http://s6.sinaimg.cn/middle/4acf0a97g7764fdec5915&amp;690" width="640" height="427"></a><br>
然而进入山区之后却出现了极低的云层，那虚无缥缈的云雾给沿途的村落蒙上了一层神秘的面纱。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s1.sinaimg.cn/orignal/4acf0a97ga9f1eb9a0860"><img name="image_operate_84261312757935881" src="http://s1.sinaimg.cn/middle/4acf0a97ga9f1eb9a0860&amp;690" width="640" height="427"></a><br>
而半山上零星的农舍几乎全被笼罩在了云雾之中。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s7.sinaimg.cn/orignal/4acf0a97ga9f1ec6feb26"><img name="image_operate_77881312757973102" src="http://s7.sinaimg.cn/middle/4acf0a97ga9f1ec6feb26&amp;690" width="640" height="427"></a><br>
期间路过一个几乎没有站台的小站，站上的二手家具饰品店竟然把商品摆放在了车站的铁路边，这样的陈设布置也不知道是有意为之还是自然的巧合，但不管怎样都让这段充满怀旧风格的旅程又增加了几分难得的意境。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s10.sinaimg.cn/orignal/4acf0a97ga9f1ecd94279"><img name="image_operate_24721312758251688" src="http://s10.sinaimg.cn/middle/4acf0a97ga9f1ecd94279&amp;690" width="640" height="427"></a><br>
到达因特拉肯之前在一片开阔地上捕捉到了这样一个瞬间，一片树林、三两户农舍、成片的草地、只闻牛铃声却不见牛的身影……以及远处浓浓的云雾，这样的画面让我想起了不知道哪部电影中的场景。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s16.sinaimg.cn/orignal/4acf0a97ga9f1ed4620df"><img name="image_operate_88161312758336458" src="http://s16.sinaimg.cn/middle/4acf0a97ga9f1ed4620df&amp;690" width="640" height="427"></a><br>
这处农舍的小院在植被的设置上有意突出了多样性的特点，小小的院落看上去竟如同微型植物园。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s7.sinaimg.cn/orignal/4acf0a97ga9f1edada646"><img src="http://s7.sinaimg.cn/middle/4acf0a97ga9f1edada646&amp;690" width="640" height="427"></a><br>
列车又停在了一个不知名的小站，列车员顺带手在车站咖啡厅买了一杯热气腾腾的咖啡。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s14.sinaimg.cn/orignal/4acf0a97ga9f1ee680d8d"><img src="http://s14.sinaimg.cn/middle/4acf0a97ga9f1ee680d8d&amp;690" width="640" height="427" name="image_operate_60891312760540390"></a><br>
沿途时常看到身背伸缩拐杖和登山包的爬山老人，其中也看到了亚洲的面孔，不过他们既不是日本人也不是中国人，而是喜欢赶时髦的韩国老人。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s9.sinaimg.cn/orignal/4acf0a97g7764fe475668"><img name="image_operate_2791312758442491" src="http://s9.sinaimg.cn/middle/4acf0a97g7764fe475668&amp;690" width="640" height="427"></a><br>
路过这片水域的时候时辰还早，湖中停放着几条小船，让宁静的湖水多了几分灵气与神秘。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s11.sinaimg.cn/orignal/4acf0a97ga9f1f107265a"><img src="http://s11.sinaimg.cn/middle/4acf0a97ga9f1f107265a&amp;690">
</a><br>
远处的天空已经有彩霞穿透了云层，多变的天气让沿途的风景更显得变幻莫测。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s15.sinaimg.cn/orignal/4acf0a97g7764fe8f3a4e"><img name="image_operate_62421312758671843" src="http://s15.sinaimg.cn/middle/4acf0a97g7764fe8f3a4e&amp;690" width="640" height="427"></a><br>
到达因特拉肯的时候天已经放晴，路旁的野花无拘无束地怒放着。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s14.sinaimg.cn/orignal/4acf0a97ga9f1f20bec4d"><img name="image_operate_42991312758757347" src="http://s14.sinaimg.cn/middle/4acf0a97ga9f1f20bec4d&amp;690" width="640" height="427"></a><br>
没想到的是火车离开因特拉肯的时候再度进入了云雾缭绕之中。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s12.sinaimg.cn/orignal/4acf0a97ga9f1f2b72bdb"><img name="image_operate_31311312758783383" src="http://s12.sinaimg.cn/middle/4acf0a97ga9f1f2b72bdb&amp;690" width="640" height="416"></a><br>
不过此时的气温已经开始上升，太阳也已升得老高。我想，天空一定会通透起来吧……<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s3.sinaimg.cn/orignal/4acf0a97ga9f1f3896da2"><img name="image_operate_22051312758901834" src="http://s3.sinaimg.cn/middle/4acf0a97ga9f1f3896da2&amp;690" width="640" height="427"></a><br>
路边的村落从车窗旁划过，远处的群山依旧被云层缠绕着。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s12.sinaimg.cn/orignal/4acf0a97ga9f1f3ff045b"><img name="image_operate_74481312758934501" src="http://s12.sinaimg.cn/middle/4acf0a97ga9f1f3ff045b&amp;690" width="640" height="427"></a><br>
在路过一个小村子时被一户农家窗台上的鲜花所吸引-实际上在这条黄金线上不仅有梦幻般的自然奇观，同样具有许多令人回味的人文景象。<br>

<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s10.sinaimg.cn/orignal/4acf0a97ga9f1f4570fd9"><img src="http://s10.sinaimg.cn/middle/4acf0a97ga9f1f4570fd9&amp;690" width="427" height="640" name="image_operate_60151312760640464"></a><br>
列车快要到达日内瓦了，此时的天空已彻底透出了蓝色。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s13.sinaimg.cn/orignal/4acf0a97ga9f1f4e6713c"><img name="image_operate_51121312759082436" src="http://s13.sinaimg.cn/middle/4acf0a97ga9f1f4e6713c&amp;690" width="640" height="427"></a><br>
面前的草地也因为光线的变换而愈发显得鲜嫩。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s6.sinaimg.cn/orignal/4acf0a97ga9f1f5637ab5"><img name="image_operate_41921312756606905" src="http://s6.sinaimg.cn/middle/4acf0a97ga9f1f5637ab5&amp;690"></a><br>
渐渐的，路边的房屋多了起来，我意识到此行的目的地日内瓦就要到了。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s3.sinaimg.cn/orignal/4acf0a97g7764ff5cdea2"><img src="http://s3.sinaimg.cn/middle/4acf0a97g7764ff5cdea2&amp;690" width="640" height="427" name="image_operate_33101312760671352"></a><br>
日内瓦湖终于展现出了她娇媚的容颜。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s13.sinaimg.cn/orignal/4acf0a97ga9f1fa18fcdc"><img src="http://s13.sinaimg.cn/middle/4acf0a97ga9f1fa18fcdc&amp;690" width="427" height="640"></a><br>
日内瓦之前最后一个不知名的小站，从站台上就能望见远处的日内瓦湖。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s3.sinaimg.cn/orignal/4acf0a97ga9f1fa7abb12"><img name="image_operate_90531312756722154" src="http://s3.sinaimg.cn/middle/4acf0a97ga9f1fa7abb12&amp;690" width="640" height="427"></a><br>
日内瓦郊区的农田从视线中划过。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s5.sinaimg.cn/orignal/4acf0a97g776500064ec4"><img name="image_operate_79531312759351864" src="http://s5.sinaimg.cn/middle/4acf0a97g776500064ec4&amp;690" width="640" height="427"></a><br>
终于到达了日内瓦，这座瑞士最为国际化的城市。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s5.sinaimg.cn/orignal/4acf0a97ga9f200a540f4"><img name="image_operate_44461312759429973" src="http://s5.sinaimg.cn/middle/4acf0a97ga9f200a540f4&amp;690"></a><br>
日内瓦著名的雕塑-断腿的椅子，这个雕塑的设置是为了表达反对在战争中使用地雷。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s13.sinaimg.cn/orignal/4acf0a97ga9f2010027ac"><img name="image_operate_96631312759377198" src="http://s13.sinaimg.cn/middle/4acf0a97ga9f2010027ac&amp;690" width="427" height="640"></a><br>
日内瓦大街上的垃圾桶被瑞士的国旗所取代，我忽然意识到瑞士的国庆节到了。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s14.sinaimg.cn/orignal/4acf0a97ga9f2017b4b6d"><img name="image_operate_47541312756796324" src="http://s14.sinaimg.cn/middle/4acf0a97ga9f2017b4b6d&amp;690"></a><br>
日内瓦市中心的建筑-这座城市并不像其他瑞士城市那样显得极其古老，从建筑形式上看反而有种强烈的现代感。<br>
<a href="http://blog.photo.sina.com.cn/showpic.html#url=http://s1.sinaimg.cn/orignal/4acf0a97ga9f201e5aa90"><img name="image_operate_44701312759585615" src="http://s1.sinaimg.cn/middle/4acf0a97ga9f201e5aa90&amp;690"></a><br>
日内瓦湖，此时天气已经彻底转晴，旅馆的服务员幽默地对我说：谢谢你给日内瓦带来了大晴天-我这才知道日内瓦也已经经过了十几天的雨雾天气了，原来我只用四个小时的旅程走出了当地十几天走不出来的雨雾天气，随后的几天我在日内瓦度过了一段阳光灿烂的日子！</p>
