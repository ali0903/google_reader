来自台湾的 @tommy351 专门为 Octopress 创作的简约风格主题 Slash。这应该是目前第一款独立发布可安装的 Octopress 主题了。

Slash 设计风格上非常简约，但 @tommy351 还是细心的加入了很多实用的功能：

  • 自动读取图片的 alt 属性，并在图片下方显示注解。
  • 内建 Fancybox，让您轻松展示您的作品。
  • 自动将 HTML5 video、iframe、object 等嵌入式影片缩放至页面宽度。
  • 支援 responsive layouts，无论电脑、手机、平板都能获得极佳的使用体验。
  • 更多介绍…

主题安装方法：

在终端中运行以下命令就能完成主题安装：

$ cd octopress
$ git clone git@github.com:tommy351/Octopress-Theme-Slash.git .themes/slash
$ rake install['slash']
$ rake generate

如果你喜欢这个主题，别忘了去作者的发布页面去表达下支持。
