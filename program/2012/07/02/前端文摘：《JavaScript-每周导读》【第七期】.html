前端文摘：《JavaScript 每周导读》【第七期】

《JavaScript 每周导读》第七期继续向大家分享最新的 JavaScript 重点资讯、技术文章、开发工具以及新出炉的各种开发框架，让 JavaScript 开发人员了解领域内的技术发展趋势，详细内容如下：

 

 

精华文章

 

FLAC.js: 纯 JavaScript 实现的无损音频压缩解码器

FLAC 是一套著名的自由音频压缩编码，其特点是无损压缩，FLAC.js 是基于 Aurora.js 框架构建的纯 JavaScript 无损音频压缩解码器。

 

关于 'null' 和 'typeof'

Kiro Risk 通过深入探究 V8 中 “typeof” 操作符的实现方式来告诉大家为什么 “typeof null” 会返回 “object”。

 

在 ECMAScript 中增加默认值操作符"??"的建议

在 ECMAScript 的 Wiki 中有一个讨论增加 '默认值操作符 (??) 的提议，以避免访问一个已丢失的属性。

 

使用 Node 和 NodeMailer 来发送邮件

这篇文章演示了如果通过 Node.js 和 Nodemailer 来发送邮件。

 

使用AngularJS 构建的 Web 应用程序展示

AngularJS 是 Google 的HTML模板项目，最近刚发布1.0版本，这个页面展示了使用 AngularJS 构建的优秀 Web 应用程序。

 

代码和框架

 

qooxdoo 2.0 发布

qooxdoo是一个创新的 JavaScript 开发框架，借助它能够开发出类似于桌面风格的 Web 应用程序和移动应用程序。

 

JSTableView: 高性能的滚动列表

JSTableView 是一个效果平滑，高性能的滚动列表插件，用于移动应用程序开发。

 

jsDraw2DX: JavaScript SVG 图形库

JavaScript SVG 图形库，功能和 Raphael 类似，用于在浏览器中绘制矢量图形。

 

Backbone UI: 封装 Backbone 模型和集合为 UI 组件

Backbone UI 提供了一种封装 Backbone 模型和集合为 UI 组件的简单方法，所有 UI 组件使用 CSS 进行外观定制。

 

Peity: 现金的 Canvas 图表插件

Peity 是一款 jQuery 插件，用于把元素的内容转换为基于 Canvas 的饼图、柱状图或者线图，效果很棒。
