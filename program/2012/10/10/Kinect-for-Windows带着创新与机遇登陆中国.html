Kinect for Windows带着创新与机遇登陆中国

10 Oct 2012

Kinect for Windows带着创新与机遇登陆中国

by 微软亚洲研究院

at 2012-10-10 09:22:47

original http://blog.sina.com.cn/s/blog_4caedc7a0102ei1b.html

编者按：2012年10月9日，一直备受瞩目的Kinect for Windows带着它独特的交互方式以及巨大的创新潜能正式进入中国市场。中国用户将感受到微软对自然用户界面开发的长期承诺，以及世界一流的销售和支持生态系统。Kinect的到来将为中国企业和开发者提供一个自由释放创造力的广阔平台，与Kinect for Windows这一不断提供成功所需的技术创新以及商业基础设施的平台合作，将给企业用户带来无限创新机遇。在技术创新的道路上，这是一个里程碑，也是一个新的起点。 

编撰：罗佳园

 

2012年10月9日，大家期盼已久的Windows版Kinect感应器正式在中国发布。同期，微软还发布了最新版本的Kinect for Windows软件开发工具包(SDK)，以及供全球下载的运行时间。Kinect for Windows在中国的发布标志着自身发展的一个新的里程碑，同时它所带来的自然人机交互界面改变日常体验的巨大潜力也将为中国企业创造全新的发展机遇。随着Kinect for Windows在中国的发布，我们相信技术领导厂商和开发者将利用这革命性平台推动时代的创新。

 

Kinect for Windows平台支持企业和开发者利用PC及其它Windows终端使用Kinect，以开发更多应用给用户带来更多创新体验。Kinect for Windows平台包括Kinect for Windows感应器，Kinect for Windows开发工具包（SDK），以及商用许可。Kinect for Windows能够帮助用户通过手势和语音与计算机进行自然交互，从而向各行业领导厂商提供全新的交互工具以改变其客户和员工的人机交互方式。

 

Kinect for Windows让人机交互的操作不再局限于使用键盘、鼠标或触控屏幕，使用者只需给出口令即可完成应用命令操作。如果他们想要在屏幕上移动对象，只需通过手势即可完成。这一创新性的自然界面人机交互迅速被引入各个领域，开启了全新的计算应用体验类别，其中包括：

 

零售行业——与零售商店和公共场所数字标牌信息亭进行远程互动；

医疗行业——充分利用语音和手势功能进行远程诊疗，以确保无菌环境和免控制要求；利用Kinect for Windows的人体跟踪功能来开发培训和模拟工具，如物理疗法应用、员工学习模块或体育健身应用；

教育领域——在教室中，对数据进行可视化和操纵。

 

Kinect for Windows感应器目前可在京东商城网站购买。Kinect for Windows体感终端提供了Windows开发者所要求的功能和特性，其中包括：用户可在无需接触键盘或屏幕的情况下进行人机交互（例如在会议室、手术室或办公室）的近距离功能。

 

Kinect for Windows软件开发工具包SDK包括先进的骨骼跟踪、最新微软语音组件、世界一流的声学模式、商业就绪的安装程序、将多个传感器插入同一台计算机的能力、帮助开发者探索体感设备和SDK全部功能的示例等核心功能特性。



最新版SDK包括以下更多功能：

 

帮助企业通过Kinect for Windows实现更多应用

可扩展的体感数据访问——包括彩色摄像头设置和扩展深度数据——这样的新功能将继续鼓励在新的、不同的地方创新地利用Kinect for Windows。

 

提高开发者效率

为了让平台对开发者来说更简单、更强大，Kinect for Windows团队发布了更多工具和示例。其中一个示例展示了Kinect for Windows人机界面指南基础上同类最佳的用户界面。

    

可扩展的Windows工具和操作系统支持

对Windows 8桌面应用、Microsoft .NET 4.5和Microsoft Visual Studio 2012的支持让企业能够在各种Windows平台上进行灵活构建和部署。

 

Kinect for Windows最新SDK开发工具包功能的完整列表和 SDK下载详见Kinect for Windows网站。

 

2010年6月Kinect正式发布，它为我们带来了全新的人机交互体验。首次登台亮相后，Kinect立即得到了广大用户的青睐，凭借发布后首两个月800万台，平均每天133,333台的销量，被吉尼斯世界纪录组织评为有史以来销售最快的电子消费产品，这个数字超过了苹果公司iPhone 和iPad产品发布后的同期销售量。用户的认可为微软研究员们带来了更大的动力，在此后两年多的时间里，他们不断释放想象的力量，让Kinect为人们带来更多交互体验的惊喜。

 

微软亚洲研究院为Kinect所贡献的技术

 

Kinect的诞生和改善离不开全球微软研究院的贡献，硅谷研究院的骨骼跟踪、雷德蒙研究院的语音识别、微软剑桥研究院之动作识别……此间，更不乏微软亚洲研究院研究员们的智慧之光。目前，多项由微软亚洲研究院发明的核心技术，如用户(玩家)身份识别、虚拟化身Avatar、实物对象数字化、脸部方向追踪、手势构建等已经成功转移进Kinect，在其不断完善的过程中发挥着重要的作用。下面请看具体的技术介绍：

 

技术一：Kinect用户身份识别（Kinect User Identity）

 

Kinect身份识别是Kinect体感设备的重要组成部分之一，它将多种技术和精心设计的用户交互整合在一起，旨在实现玩家身份识别与跟踪的目标，帮助实现流畅、自然的用户界面。主要识别方式为：

 

生物识别登陆(Biometric Sign-in)，系统经过一段时间，了解特定玩家的外形，并且当此人出现在视野中时自动登录；

会话(Session)，系统会在特定游戏会话过程中记住谁是谁，例如，玩家1与玩家2的对抗，以及他们各自的得分。

 

技术二：虚拟化身（Kinect Avatar）

 

Kinect Avatar是在Xbox 360等设备中为玩家预设的角色化身。这些角色化身虽是虚拟的，但其表情、姿势、动作却可以说是真实的。在游戏中，Kinect能细致捕捉用户整个身体的动作和姿势，并迅速投射到游戏里的虚拟角色身上。你跳，游戏里的Avatar也跳；你出拳、踢腿，游戏里的Avatar也照做。更妙的是还能进行个性化交流的时候，Avatar会将用户脸上所有的表情特征表达出来，你可以看到自己的一颦一笑。

 

技术三：实物对象数字化（Kinect Based Object Digitization）

 

实物对象数字化技术可以在对物体扫描快照的基础上创建3D对象。结合Kinect深度相机捕捉，物体的正面和背面的色彩和深度图像迅速在二个维度进行注册，而基于GPU的表面重建则形成一个平滑的3D模型。在Kinect前，任何有形物体都可以在转瞬间实现数字化，并以3D方式展示。

 

此技术大体可阐述为：首先通过分割，从所捕捉图像中删除非目标部分；然后利用摄像头数据创建的点云；之后在点云之上适配3D表面，并对最终的数字化对象进行风格化渲染和动画；最后系统合并图像，并把它们转换成代表着对象的正面和背面表层的3D点区域。 

 

Kinect技术引出了许多新兴研究领域，包括人体追踪、人体和物体的数字化等。研究者们在不断探寻一条更佳的路径去跟踪人们身体运动、手势和表情等，也在创造性地思考着如何通过扫描物体来制作更为精确的数字模型。微软亚洲研究院在这些方面的努力虽已取得了显著成果，但这只是探索的开始。希望Kinect在登录中国后，我们能在这条路上走得更远。当然，我们也相信不断自我完善的Kinect 将为广大开发者们提供更多的可能性，让热爱Kinect的开发者们在这一平台上创造出更多创新成果。
