iTools 2012 Beta 0329
新增The new iPad的全面支持，包括桌面管理、壁纸等
新增对iOS 5.1的支持，优化图片管理、电子书库等



新增网络音乐资源“多米音乐”可以试听、下载到正版的音乐


新增本地“媒体库”，加入音乐、壁纸、铃声



新增Apple ID 验证，支持用户使用正版升级
新增Apple ID 注册教程
新增正版升级、第三方升级、一键智能升级


新增联系人分组中“未分组”，加入键盘上、下操作支持
新增更新日志
新增IPA关联功能及选项，可以双击安装IPA文件及在网络上进行“iTools一键安装”


新增网络资源的本地缓存
新增HTTP网络代理功能
新增修复沙盒引导
修复iOS5下短信对话无法以联系人名字显示的问题
图片悬停时显示更多信息
优化并修复图片管理的体验及BUG
修复部分用户使用图片浏览器时无法截图的问题
大幅提高图片导入速度
