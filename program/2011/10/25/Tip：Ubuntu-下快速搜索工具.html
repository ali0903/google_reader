本文，兔兔给大家总结下Ubuntu 或其他Linux 发行版本下文件快速搜索工具、文件预览工具。

文件搜索工具

#1，GNOME DO

-方便快捷地处理你的文件、书签、应用程序、音乐、通讯簿……还有很多！

-安装链接：http://apt.ubuntu.com/p/gnome-do

#2，Synapse

-类GNOME Do 工具

-安装：http://apt.ubuntu.com/p/synapse

文件预览工具

#1，GNOME 3.2 Sushi

安装：

sudo apt-get install gnome-sushi

#2,Gloobus Preview

Ubuntu 11.10 可使用下面命令，用Gloobus Preview 来代替NGOME Sushi：

sudo add-apt-repository ppa:gloobus-dev/gloobus-preview
sudo apt-get update
sudo apt-get install gloobus-preview gloobus-sushi
