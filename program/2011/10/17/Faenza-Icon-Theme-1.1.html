Faenza Icon Theme 1.1

by riku

at 2011-10-17 09:23:12

original http://wowubuntu.com/faenza-11.html

Linux 平台上最流行的图标集之一 Faenza Icon Theme 发布 1.1 版，主要增加了对 GNOME 3.2 的支持，新增了很多新程序的图标，如 Gnome Documents, GNOME Contacts, GNOME Online Accounts 及  Clementine, Xterm, Wunderlist, Scribes, Steam, Tracker 等等。

另外，此版本的另外一个重要更新是，提供与 Ubuntu 11.10 Unity 界面更好的整合效果。

下载： Gnome Look

# Ubuntu 用户安装

sudo add-apt-repository ppa:tiheum/equinox
sudo apt-get update
sudo apt-get install faenza-icon-theme
