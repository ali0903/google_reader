Quick Batch File Compiler 简体中文版 bat2exe 软件

Quick Batch File Compiler 是一个批处理转exe 软件，它可以将你编写好的任何批处理文件（.bat/.cmd）编译成 exe 可执行程序。并将其加密以防止反向解包。，Quick Batch File Compiler还允许你为exe文件添加公司名称、软件版本号、应用程序图标等个性化设置。
