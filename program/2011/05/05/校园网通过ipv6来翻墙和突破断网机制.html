校园网通过ipv6来翻墙和突破断网机制

by GFW BLOG 功夫网与翻墙

at 2011-05-05 00:26:53

original http://feedproxy.google.com/~r/chinagfwblog/~3/uQELvUwpq8Y/ipv6.html

来源：http://zyowong.blogspot.com/2011/04/ipv6.html

首先，从校园网中获取ipv6的IP。
1.1xp用户，打开命令行窗口输入ipv6 install安装ipv6协议，如果现在打开http://ipv6.google.com.hk，可以访问的话，证明人品够高，已经成功配置了。但是如果不行的话，继续执行以下步骤，建立隧道连接。

在窗口依次输入netsh,interface,ipv6,isatap,并逐个回车，最后输入set router 192.168.40.1 //可以参考win7的做法
1.2win7用户，因为已经预安装了ipv6协议，以管理员身份运行命令行窗口建立隧道连接。
netsh(回车，输入大于符号以后的内容，下同)
netsh>int
netsh>interface>ipv6
netsh>interface>ipv6>isatap
netsh>interface>ipv6>isatap>set router isatap.scu.edu.cn
netsh>interface>ipv6>isatap>set state enabled
netsh>interface>ipv6>isatap>quit
2.设置代理ipv6的代理edu6.zzzcn.info:2012，可以翻墙并突破断网。
3.突破校园内的断网，是由于校园取消了DNS的解析，所以只要在DNS那里填上海交大DNS服务器：2001:da8:8000:1:202:120:2:101，就可以继续用IPV6上网了。
PS.再提供一个上交的router地址，如果华农的那个用不到就试试这个吧，希望校园网翻墙不再是梦。
set router isatap.sjtu.edu.cn
