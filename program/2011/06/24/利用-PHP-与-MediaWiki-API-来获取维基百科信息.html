维基百科是终极在线百科全书，其中收录了几百万条各类人文知识。Web 应用程序开发人员能够通过 Wikipedia API 来访问并搜索这些条目。本文介绍了 Wikipedia API，并在 PHP 应用程序环境中对其进行了演示，来说明如何使用 PHP 来搜索和检索维基百科内容的不同元素。
