TCP Optimizer下载链接：TCPOptimizer.exe

下载直接运行即可开始设置。

Windows Xp设置步骤

参照截图，先将下方选为“Custom”，再勾选“Don’t Modify Network Adapters”，再将上方的Connection Speed拉到至少20Mbps，再将大红框内的选项设置得与跟截图中一样，最后点最下方的Apply，这时会弹出提示，再次确定，设置完成后软件会提示重启计算机。

Windows 7/Vista设置步骤

参照截图，先将下方选为“Custom”，再勾选“Don’t Modify Network Adapters”，再将上方的Connection Speed拉到至少20Mbps，再将大红框内的选项设置得与跟截图中一样，最后点最下方的Apply，这时会弹出提示，再次确定，设置完成后软件会提示重启计算机。

原文：http://www.ignw.net/doc/windows-tcp-optimize.html

作者说Windows系统的默认网络性能十分糟糕，使用文中方法优化之后可以大幅提高vpn速度，本人未测试。
