还记得去年那篇LOGO设计的10个发展趋势么？今年的《2011年Logo设计趋势》也在早些天发布了，国外著名的Logo综合网站Logolounge.com每年都会发布当年的Logo设计趋势报告，今年也不例外。LogoLounge在报告首行即指出今年的Logo设计趋势的主题是：“去年的更明亮、今年的更简洁”。

2011年Logo趋势01：渐变

例如： 1. Rylander Design, Baker Ave 2. Pixonal, Stallion 3. Signifly, Plesso 4. Chermayeff & Geismar Inc., Women’s Tennis Association

2011年Logo趋势02：卡通化

例如：1. Design Ranch, Target 2. Schakalwal Design Studio, Musiq Launch 3. Vectory Belle, Box of Cravings 4. Tad Carpenter, Yummo Yogurt and Smoothies

2011年Logo趋势03：线条

例如：1. Vistaprint, Vistaprint 2. FCB Durban, Oubaai 3. RDQLUS Creative, Expeditiously Delicious 4. Sommese Design, Dantes Restaurants

2011年Logo趋势04：系列Logo

例如：1. Gardner Design, College Hill Neighborhood Association 2. Hulsbosch, Woolworths Limited 3. Sunday Lounge, Weathervane Farm 4. Sebastiany Branding & Design, Softway

2011年Logo趋势05：棕色

例如：1. 1310 Studios, Small Hands Big Art 2. Gerren Lamson, NONA 3. Defteling Design, Candywinkel 4. Odney, MBT’s

2011年Logo趋势06：同轴心

例如：1. Notamedia, gogol.tv 2. Sommese Design, State College State Theatre 3. Clover Creative Group, LLC, CLOVR Media Inc. 4. Vectory Belle, Web Application

2011年Logo趋势07：线圈

例如：1. Noetic Brands, Green Bills 2. Shawn Wideman, Vida Church 3. petervasvari.com, ACTUART 4. Kraftwerk Design Inc., Terravant Wine Company

2011年Logo趋势08：丝带

例如：1. BrandBerry, Artive 2. Higher, Voscast 3. Dickerson, Healing Touch 4. Frog Design, Microsoft Word

2011年Logo趋势09：符号

例如： 1. FutureBrand BC&H, Grupo Boticário 2. Karl Design Vienna, Bertha Benz Challenge 3. BrandBerry, Olive 4. Baris Atiker, Euroclean

2011年Logo趋势10：色基球

例如： 1. Siren Design, Berkeley Estates 2. Vladimir Isaev, Telecom trade company 3. Dee Duncan, Razorfish 4. Publicity Upstream, Nouveaux Marches de France
