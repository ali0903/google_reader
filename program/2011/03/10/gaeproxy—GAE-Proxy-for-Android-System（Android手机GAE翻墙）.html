GAppProxy http://code.google.com/p/gappproxy/ Android 客户端，综合 TransProxy、CMWRAP、SSHTunnel 和 Python For Android 等项目，以实现 Android 系统上基于 Google AppEngine 的 HTTP 代理。

注意

GAEProxy 新版本发布，解决了众多关键问题，推荐更新。部分用户升级后需要在关闭服务状态下，在主界面按menu键，选择 “清除连接状态”按钮，以清除过期的 DNS Cache。

准备工作

1. 部属自己的 GAppProxy Fetch Server ,详情请见：http://code.google.com/p/gappproxy/wiki/GAppProxy_2_0_0_Manual

2. 下载并安装 GAEProxy 最新版。

3. 第一次运行 GAEProxy 请按照提示进行设置，并确保在联网并插入SD卡的状态下勾选 “安装依赖模块” 。

4. 现阶段 GAEProxy 的运行依赖于 iptables ，并需要获得 ROOT 权限。因此请确保 “自动设置代理” 选项开启。

5. 勾选 “开启服务”，浏览互联网。
