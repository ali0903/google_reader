向日葵 1.0 Beta3发布 在家远程控制办公室电脑

original http://www.cnbeta.com/articles/138018.htm

工作事情太多，每晚加班太晚，家人心生怨念，还折腾了自己的胃，别愁，让向日葵远程控制软件来帮你。整个过程完全无视任何内网阻碍，再也不用求网管。这使用向日葵只有简单三步。

1.     注册oray 护照

登录 https://console.oray.com/passport/register.html 简单几步注册人个护照；


2.     被控端(大C的办公电脑)安装向日葵客户端并登录向日葵

登录 http://www.oray.com/sunlogin/download.php 下载3.0beta版向日葵客户端 - 3.0 版向日葵支持 VPN、RDP协议、云配置、实时动态屏幕监视墙等等，比 2.x 版功能更强大，更稳定。

安装向日葵 3.0 客户端。

VPN驱动，点击“安装”继续安装剩余软件

第二步，向日葵已成功完成安装啦！

 

接着是运行并登录向日葵 3.0 客户端(被控端)

 

第一次运行向日葵客户端时会提示登录

输入刚才注册的 Oray 护照及密码登录向日葵。

选择 Oray 赠送的永久授权登录

登录后系统右下角托盘区会显示一个向日葵花图标。

 

3.     使用任意 PC，开始远控

 

使用浏览器进行远控：

 

使用任意 PC ，打开 IE 或 Firefox 并用刚才注册的护照登录 Oray 向日葵管理中心

https://console.oray.com/sunlogin/remote/


 

登录后即可看到

点击右侧的”远控”按钮即可开始远控

 

输入windows 系统用户名(支持域用户名)及密码，回车

 

选择”远程桌面”

主控客户端进行远控：

 

下载向日葵主控端

 

登录 http://www.oray.com/sunlogin/download.php 下载 “向日葵主控端1.0体验版”

之后运行安装，安装过程同客户端一致。

安装后运行主控端，界面如图所示：

怎么样，是否很简单，向日葵远有很多功能，等着你一一去探究。
