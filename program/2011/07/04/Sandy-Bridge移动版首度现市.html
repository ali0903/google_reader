Sandy Bridge移动版首度现市

by

at 2011-07-04 11:42:27

original http://www.cnbeta.com/articles/147722.htm

和桌面产品不同，笔记本处理器一般都是提供给OEM PC厂商预装使用，零售市场上并不多见，而且DIY升级难度也比台式机大得多。Sandy Bridge架构面世整整半年之后，我们终于在市面上第一次看到了零售版的移动型号。

这三款零售上市的移动版Sandy Bridge分别是四核心的Core i7-2720QM和双核心的Core i5-2540M、Core i5-2520M，均使用Socket G2封装接口(又称为PGA988B)，需搭配6系列移动芯片组，不兼容上代Arrandale Core ix系列所使用的Socket G1(又称为PGA988A)和5系列芯片组。包装盒侧面的“Requires Socket G1”字样请无视。

除了用于给笔记本升级之外，这些移动处理器也可以搭配一些采用6系列移动芯片组的主板，比如iBASE的MI956(QM67芯片组)、Supermicro的Mini-ITX迷你型X9SCV-Q(QM67芯片组)等等。可惜的就是这类产品太少，而且大多面向专业领域，零售市场上不容易找到。 

Core i7-2720QM是一款四核心八线程型号，主频2.2GHz，加速最高3.3GHz，三级缓存6MB，集成图形核心HD Graphics 3000，动态频率650-1300MHz，支持双通道DDR3-1600内存，最大容量16GB，各种处理器、图形技术一应俱全，热设计功耗45W。

Core i5-2540M是双核心四线程，主频2.6GHz，加速最高3.3GHz，三级缓存3MB，图形核心同上，内存降为8GB DDR3-1333，相关技术一样不少，热设计功耗35W。

Core i5-2520M就简单了，原始频率和加速频率分别降至2.5GHz、3.2GHz，其它完全同上。

三者的S-Spec编号分别确认为SR014、SR048、SR044。

Core i7-2720QM、Core i5-2540M/2520M官方标价分别为378美元、266美元、225美元，目前在日本秋叶原市场上报价分别为34480日元、24980日元、20480-20980日元，约合人民币2760元、2000元、1640-1680元。
