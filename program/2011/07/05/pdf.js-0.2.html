pdf.js 0.2

by 黑日白月

at 2011-07-04 17:46:18

original http://linuxtoy.org/archives/pdfjs-02.html

使用 JavaScript 编写的 PDF 解析器发布 0.2 版本，实现Type 1 字体加载。

同时还实现了：

  • 浓淡图像。
  • 混合字型。
  • 左侧预览视图。
  • 连续页滚动显示。

目前该功能在 Firefox Nightly 效果完美，其他浏览器亦可运行，但兼容性方面尚需改进。
