5个在线代码编辑器，供云计算爱好者

by

at 2011-04-07 10:18:46

original http://news.cnblogs.com/n/96583/

随着云概念的流行，云计算的商业化的深入，云的使用者或是爱好者越来越多。本文精选了几款在线代码编辑器，开发者可以在浏览器中进行工作。

我们要多长时间才能完全以云的方式来编程呢？先看看这5个在线代码编辑器吧，如果你是开发者，如果你关注云计算。

Ace / Cloud 9

URL: http://cloud9ide.com/

URL: http://ace.ajax.org/

完全是Javascript编写，基于浏览器的代码编辑器，Ace是个成功者。 Bespin, 基础代码已经整合到了Ace。这是个开源项目，托管在GitHub。Cloud 9 同时提供商业版和免费版，允许多个开发者远程协作开发。

Kodingen

URL: http://kodingen.com/

Kodinegen 是个CDE，或基于云的开发环境。开发者可以越过传统的服务器设置、配置、或其他开发环境相关的头疼的事儿，轻松、快捷的运转项目。Kodingen 很是不同，几乎所有的东西都托管在云端。它同时支持通过 Pixlr and Aviary 进行图片编辑。源码控制支持SVN，GitHub及其他代码托管平台。

BuildorPro

URL: http://buildorpro.com/

位于伦敦的初创公司BuildorPro意识到，作为前端设计师和开发者最终要部署到web，为什么不直接将工作流、代码创建放到浏览器中呢？它和dreameaver比如何？你可以try一下。

CodeRun

URL: http://www.coderun.com/

CodeRun 即时基于浏览器的IDE，也是代码分享平台。尤其针对.net开发者，Visual Studio的界面在这里也能找到。

支持.NET 3.5 (C#), PHP (5.1), JavaScript, HTML and CSS，C# 支持包括ASP.NET, WCF, Silverlight ,WPF 浏览器应用开发和部署。数据库支持包括 SQL Server 2005 和 Amazon SimpleDB。

PHP Anywhere

URL: http://phpanywhere.net

PHP Anywhere 允许你几乎支持在所有的浏览器上开发PHP应用。支持在iPad上直接编程。支持 PHP 5 ，代码编辑器支持语法高亮等。
