CSS3Menu 是一款制作网页导航菜单的工具，并且导出标准的 html + css 文件。@Appinn

拥有漂亮的 CSS 导航菜单，是很多同学都希望的。CSS3Menu 能帮助你十分简单并且省事的创建一个导航栏菜单，你只需点几下鼠标，选几种颜色，完事。如果利用内置的模板那就更容易了，只需输入菜单上的文字即可。

比较遗憾的是在 @scavin 试用过程中，CSS3Menu 的导出功能貌似不支持 UTF-8，也就是说你需要手动修改代码，添加中文。不过对于习惯了手写 CSS 的设计同学来说小菜一碟了。你可以从这里看到 @scavin 随手制作的一个导航栏。
