Project ROME 是款 Adobe AIR 平台的内容编辑平台，十分华丽，内置模版也很丰富。@Appinn

你可以用 Project ROME 很容易的创建出漂亮的页面，用鼠标进行页面布局，添加多媒体材料，甚至可以添加 Google Maps 在页面内。支持中文显示，但不支持输入，使用复制粘贴大法解决问题。

Project ROME 提供 AIR 平台以及 Web 版本，无需下载就能体验。另外支持导出 PDF 及 jpg 格式，略现局限。用 Project ROME 做一些漂亮的宣传册子还是很不错的。

http://rome.adobe.com/index.html
