Firefox 32个必备插件

by GFW BLOG 功夫网

original http://feedproxy.google.com/~r/chinagfwblog/~3/uX_47T848zw/firefox-32.html

Adblock Plus 1.2.2 (必备)
广告已成往事！

AutoProxy 0.4b1 (必备)
撞墙已成往事！

BetterPrivacy 1.48.3 (必备)
BetterPrivacy 是一个不错的Cookie保护屏障，避免因Cookie未被删除被追踪到痕迹。它可以自动或有选择性的删除Flash播放痕迹，浏览器中的ISO文件将时刻受到BetterPrivacy的监视。

BugMeNot 2.2
安装BugMeNot成功后，你会发现鼠标的右键菜单多了一个选项[Login with BugMeNot]，选择它，就会把你的用户名和密码"记录"到BugMeNot。用的时候，重复操作一次就可以了。

CookieSwap 0.5.253 (必备)
Cookie切换，实现多帐户登录。

Copy Link Text 1.4.9
右键复制链接的文字。

Delete Bookmark Icons 0.4.1
删除收藏夹的图标，是收藏夹看上去很干净清爽。

Download Manager Tweak 0.9.2
下载插件

DownloadHelper 4.8
从多个站点下载视频和图片文件。

DownThemAll! 1.1.10 (必备)
必备的下载插件。

FlashGot 1.2.2 (必备)
允许调用外部下载软件的单个或批量下载任务插件。

FoxClocks 2.6.18
在应用程序的状态栏/工具栏显示世界多个地域的时间。

FoxyProxy Standard 2.22.1 (必备)
卓越的代理服务器管理工具。

GProxy Tool 1.64 (必备)
配合自由门、无界代理、花园代理等一些自动代理的插件，翻墙必备插件。

Greasemonkey 0.8.20100408.6 (必备)
面向 Firefox 的用户脚本管理器。

IDM CC 7.1.7
调用Internet Download Manager外部下载软件。

IE Tab 2 (FF 3.6+) 2.5.10.1
让 Firefox 也能内嵌 IE 用双核心!!!

Mason 0.3.7.8
支持脚本，类似于Grease Monkey。
但是区别在于，加载在DOM或者文件的最前方，执行效率更高，使用更加灵活，功能更加强大。
请访问维护站点下载更多Mason脚本!
2. 支持Http请求的重定向和阻止
可以随心所欲地阻止或者重新指定Http请求的地址
3. 让Firefox正确识别下载文件
4. 把文件同步缓存到指定文件夹
5. 修改cookie
6. 修改referrer(自定义或者伪装成站点根节点)
7. 伪装IE6
8. 支持Force content-type 以及 Refcontrol的规则导入

Menu Editor 1.2.6 (必备)
能够个性化工具栏菜单、右键菜单等的插件，仅限于windows。

NoScript 2.0.3.2 (必备)
为您的 Firefox 提供额外保护: NoScript 只允许在您选择的信任域(例如您的家庭银行网站)上执行 JavaScript、Java(和其他插件)。该白名单基于抢先阻止机制，在不损失
任何功能的前提下，防止利用已知或未知安全漏洞的攻击。专家们同意:拥有 NoScript 的 Firefox 更安全 :-)

Organize Status Bar 0.6.5 (必备)
组织您自己的状态栏图标。

Personal Menu 4.3.2 (必备)
打造自己的菜单，省去多余的空间！

PhZilla 3.5.2B
通过在线代理服务器(PhProxy)匿名的网上浏览.

ProxyBar 1.1 (必备)
在状态栏添加一个代理工具栏，通过她可以直接设置代理，很方便。

QuickRestart 1.1.6
添加"重启 Firefox"到"文件"菜单

RapidShare DownloadHelper 1.0
RapidShare 辅助下载插件。

Stop Autoplay 1.0 (必备)
禁止自动播放网页内嵌音乐和视频。

Stylish 1.0.11
让网页和用户界面有个新形象

Tab Mix Plus 0.3.8.4 (必备)
加强标签页浏览功能。

Tamper Data 11.0.1
Tamper Data 简单易用，功能强大，可以用来查看和修改 HTTP/HTTPS 的头部和 POST 参数；可以用来跟踪 HTTP 请求和响应并记时；可以对 WEB 站点进行某些安全测试，从而为调试 WEB 配置带来了极大的便利。

Text Link 3.1.2009110201 (必备)
允许通过双击载入网页中的 URI 文本。

WorldIP 2.0.2
浏览网页的同时，可以查看到对方的IP。
