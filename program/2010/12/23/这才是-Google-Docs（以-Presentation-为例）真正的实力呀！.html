2010 年就要过去了，不来点技术展示，怎么对得起广大的 Google 服务使用者呢？没错！Google 最近就丢出了一个名叫 Google Demo Slam 的活动，让大家上网参观残酷舞台，并且投下神圣的一票。

当中看来颇值得大家注意的示范之一，则是上面（影片在跳转后）这组由三位分居三地的动画师，耗时三天，仅开一个 Google Docs 文件（Presentation）就完成的旷世巨作，而预设跟这 Google Docs 动画 PK 的，则是由网坛正妹莎娃领衔主演的 Voice Search 示范，不过目前看来死会 正妹威能还是远大于这三位动画师的努力。

如果看完该影片，对于该投影片兴趣更浓厚的话，不妨到这来看个仔细。
