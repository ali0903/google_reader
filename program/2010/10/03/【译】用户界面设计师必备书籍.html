【译】用户界面设计师必备书籍

by ISLAND

at 2010-10-03 14:07:02

original http://ucdchina.com/snap/8092

想为你的藏书室进新货？那千万别错过这几本书。毫不夸张的说，它们就躺在我的书桌上，从上到下整齐的列在那里。

The Visual Miscellaneum: A Colorful Guide to the World’s MostConsequential Trivia

作者:David Mccandless

你是Edward Tuffe的粉丝？那这本书你需要；你从没听说过这个人？那这本书你需要；包括他所有的书。

这本书里，充满了各种意趣横生的可视化数据的点子。这本书在我这躲书的最上面，因为我刚知道它，噢，非常的喜欢。

Prototyping: A Practitioner’s Guide

作者：Todd Warfel

出版社：Rosenfeld Media

时间：11/2009

完全改变了我们行事的方式。充满里运用各种流行的工具进行原型设计的实例。这本书放在第二位置，因为我早上的时候在FW上制作原型的时候在本书上查看一些信息。

The Smashing Book

由Smashing Magazine独家提供，该书关注的领域有：Web设计实战规则，颜色理论，可用性指南，用户界面设计等，最佳编码和优化实践等，以及来自于世界顶级设计师关于排版，市场营销和品牌的独到眼光和见解。当然还包括我的一个Otherbox，就在第一章！

Designing Web Interfaces:Priciples and Patterns for Rich Interactions

作者:Biil Scott and Theresa Neil

出版社：O’Reilly Media

时间：01/2009

是呀，我知道这是我们出的书，但是它现在确实躺在我的桌子上。我至少每周需要查看一次。这星期刚开始，我查看了一些ZUIs，可缩放的用户界面等。

Designing for the Social Web

作者：Joshua Porter

出版社：New Riders

时间：05/2008

非常非常值得一读，买一本吧，不会浪费你一分钱的书。很多实用的建议和忠告。虽然我不用每天查看该书，但是我至少每星期推荐个五个初创公司。

Designing Social Interfaces：Principles，Patterns，and Practices for Improving the User Experience

作者：Christian Crumlish，Erin Malone

出版社：O’Reilly Media

出版日期：09/2009

“这本书详尽到位的记录了如今绝大多数的UI模式，整合了社交网络的网站之中。每个模式都有非常有趣的讨论，什么时候使用？哪些人使用？”——亚马逊的简介。我要从这堆书把它抽出来，因为在下一周的时间里，他将伴我左右。

Web Form Design：Fill in the Blanks

作者：Luke Wriblewski

出版社：Rosenfeld Media

出版日期：05/2008

每个搞网页设计的人都需要这本书！不用再去思考如何进行表单设计了，因为这么书是如此的完美。我可以做更加有趣的设计挑战。这本书不会离开的书桌。

Defensive Design for the Web：How t inprove error messages,help,forms,and other crisis points

作者：Matthew Linderman and Jason Fried

让37signals团队告诉阻止用户犯错的最佳方式，并且指导用户如何处理已经出现的错误。

About Face 3：The Essentials of Interaction Design

作者：Alan Cooper

出版社：Wiley

出版日期：05/2007

在你打破之前必须学会的规则。Pretty please with a cherry on top?不想设计那种简单的网站，那就读这本书吧。对Flex开发者Ajax开发者也非常有用。很多模式都可以RIA中推广使用。

源地址：Essential Books for User Interface Designers
