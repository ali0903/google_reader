作为 Web 设计者，我们希望实现鹤立鸡群的设计，要做到这一点，要开阔眼界。欧美同东方的 Web 设计很不同，因为属于不同的文化。韩国不仅为我们拍摄了很唯美的情色电影，他们的 Web 设计也很唯美。本文精选了35个非常漂亮的韩国 Web 设计，也许可以带来不同的感觉。
